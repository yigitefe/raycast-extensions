# Leader Key

Search Leader Key configuration for shortcuts.

## Leader Key App

[LeaderKey: The *faster than your launcher* launcher](https://github.com/mikker/LeaderKey)

## Features

- **Search Shortcuts**: A fuzzy-searchable list of all your configured shortcuts.
- **Configurable Path**: Set your own configuration file path in preferences.

## Configuration

The extension looks for Leader Key's JSON configuration file.

**Default Path:** `~/Library/Application Support/Leader Key/config.json`

## Preferences

- **Leader Key Config Path**: Change the location where the extension searches for your `config.json`. Supports `~` and `$HOME` variables.
