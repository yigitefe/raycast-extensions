# Chinese Character Converter

Auto convert clipboard content between Traditional and Simplified Chinese.

## Features

- **Smart Detection**: Automatically detects if the text is Simplified or Traditional Chinese.
- **Bi-directional Conversion**: Converts Traditional to Simplified and vice versa.
- **Privacy Focused**: All conversions happen locally on your machine using `opencc-js`.
- **Clipboard Integrated**: Reads from clipboard, converts, and copies the result back automatically.

## Usage

1. Copy any Chinese text to your clipboard.
2. Open Raycast and run the **Chinese Character Converter** command.
3. The converted text will be copied to your clipboard, and a HUD notification will show the result.
