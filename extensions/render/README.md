# Render

This extension allows you to view your Render services, deploys, domains, and more.

## Features

- View all your Render services grouped by owner
- Pin frequently used services to the top for quick access
- View deploy history and details
- View environment variables and custom domains
- Open services directly in Render or their Git repository

## Setup

To use the extension, you'll need to provide a personal API key. You can read more about creating and using API keys in [Render docs](https://render.com/docs/api#getting-started).
