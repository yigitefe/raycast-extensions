# Hevy

Your Hevy workout data at your fingertips via Raycast.

**Note:** This extension requires a Hevy API key, which is only available to Pro users. You must have a Hevy Pro subscription to use this extension.

You can find instructions on how to get your Hevy API key in the [Hevy API Docs](https://api.hevyapp.com/docs/#/Routines/get_v1_routines).
