# 0x0

Raycast extensions for the [0x0.st](https://0x0.st) file sharing service.

In its current form, the extensions is meant for simple and quick file sharing with no extended 0x0 features.

Special thanks to the amazing contributors behind 0x0. You can view their repo [here at mia/0x0](https://git.0x0.st/mia/0x0).

Created by [pseudobun](https://github.com/pseudobun).
