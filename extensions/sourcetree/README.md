# Sourcetree

Search and open [Sourcetree](https://www.sourcetreeapp.com/) repositories.

## Configuration

- **Sourcetree command line tools**: The path to the Sourcetree command line tools. If you have installed Sourcetree via Homebrew, the path is `/opt/homebrew/bin/stree`.
- **Sourcetree plist file**: The path to the Sourcetree plist file. If you have installed Sourcetree via Homebrew, the path is `/opt/homebrew/Caskroom/sourcetree/latest/Sourcetree.app/Contents/Resources/sourcetree.plist`.
